# Frontend for the test task

DEV: http://localhost:3000

API: https://swapi.dev/

## Deploy

```shell script
git clone https://github.com/A-Kazakova/Skunexus-Start-Wars-Recruitment.git
cd ./Skunexus-Start-Wars-Recruitment
yarn install
```

### Launch

```shell script
yarn start
```

### Build

```shell script
yarn build
```
